# Meituan Paotui

This plugin is a Codex store adapter for the official Meituan Paotui Skill source.

Official sources checked on 2026-06-03:

- GitHub: https://github.com/meituan/MT-Paotui-For-Client
- ClawHub: https://clawhub.ai/meituan-tech/mt-paotui-for-client

This adapter does not redistribute the upstream Meituan code. It exists so the store can list and route the capability while keeping licensing and authorization boundaries explicit.

Use this for Meituan Paotui runner-service workflows, not ordinary food delivery ordering.
